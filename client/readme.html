How to use Election Impact API?
Step 1:

Each company has assigned one API key to identify which company data to retrive. You can get your API key from adminpanel.
Step 2 :

Election Impact allows to make REST api calls to web services.

Endpoint URL: http://www.electionimpact.com/client/get_data.php
Parameters: API allows following parameters which you can specify while making a call:
Parameter Required Default Purpose
 
apikeyYes blank This is the api key which you get from admin to access (Note: This parameter is case sensitive)
searchname No blank This will filter data displays only the data that meet criteria that you specify
state_code No blank This will filter data on given State Code. This is pre-defined two digit state code of USA.
reg_source No blank This will filter data on given Registation Source.
regdateTo No blank This will filter data on given registration Date.
regdateFrom No blank This will filter data on given registration Date starting date is regdateTo and till regdateFrom
start No 1 You can specify starting record no. If you will not specify it will start from 1.
noofrecord No 500 You can specify per page record. By default, it will retrive 500 records per call. You can specify value from 1 to 500. If you specify larger value than 500, then it will fetch 500 records per call (for better performance). If you specify smaller value than 1, then it will fetch 50 records per call.
Examples of API call : http://www.electionimpact.com/client/phpsample.php
